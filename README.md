# MsPAC
Phase reads, assemble haplotypes and detect SVs

### Installing
```
git clone git@github.com:oscarlr/MsPAC.git
cd MsPAC
conda env create -f environment.yml 
conda activate MsPAC
cd ..
git clone git clone https://github.com/oscarlr/cluster
```

### Running the tests
```
cd testing
sh run.sh
```
